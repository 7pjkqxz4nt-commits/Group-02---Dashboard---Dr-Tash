# 🌟 OSHE Master – HSE AI Dashboard

A professional, AI-powered dashboard for Health, Safety & Environment (HSE) KPI monitoring and hazard analysis.  
Developed as part of the **OSHE Master Project** under the supervision of **Prof. Dr. [Name]**, University of Alexandria, Egypt.

---

## 🧠 Features
- Flexible data ingestion (CSV/Excel)
- KPI calculations (OSHA TRIR, LTIR, RIDDOR, Canadian OHS)
- Interactive charts (bar, line, pie, heatmap, risk matrix)
- AI Insights (trend detection, predictive modeling)
- Natural language Q&A (“Show me incidents in Port Said last year”)
- PDF report generation (bilingual Arabic/English)
- Secure login and corporate branding

---

## 🖥️ Tech Stack
| Component | Technology |
|------------|-------------|
| Frontend | Streamlit |
| Data Processing | Pandas, NumPy |
| Visualization | Plotly, Seaborn, Matplotlib |
| AI Layer | Scikit-learn, Transformers |
| Reporting | ReportLab |
| Deployment | GitHub + Streamlit Cloud |
